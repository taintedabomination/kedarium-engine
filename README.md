# Kedarium Engine

## Dependencies

- OpenGL
- GLFW
- GLEW
- GLM
- libPNG

## Controls

| Key        | Functionality    |
| ---------- | ---------------- |
| W          | Move forward     |
| A          | Move left        |
| S          | Move back        |
| D          | Move right       |
| Space      | Ascend           |
| Left Shift | Descend          |
| E          | Lock cursor      |
| Escape     | Unlock cursor    |
| C          | Display vertices |
| V          | Display edges    |
| B          | Display faces    |
